##  **Getting Started**  

###  **1. Clone the repository**  
```sh
git clone https://github.com/your-username/imdb-clone.git
cd imdb-clone
```

###  **2. Install dependencies**  
```sh
npm install
```

###  **3. Run the app**  
```sh
npm start
```
The app will run at **`http://localhost:3000`**
